# ovO Windows 使用说明

## 适用对象

这份说明用于把 ovO 桌面版交给 Windows 电脑使用。推荐 Windows 10/11 64 位系统。

## 生成 Windows 安装包

在项目根目录执行：

```bash
npm install
npm run dist:win
```

打包完成后，文件会生成在：

```text
release/windows/
```

默认会生成两类文件：

- `ovO-0.1.0-x64-setup.exe`：安装包，推荐发给普通用户。
- `ovO-0.1.0-x64-portable.exe`：免安装版，适合临时测试或没有安装权限的电脑。

如果只想生成安装包：

```bash
npm run dist:win:installer
```

如果只想生成免安装版：

```bash
npm run dist:win:portable
```

## 发给 Windows 用户

推荐发送 `release/windows/ovO-0.1.0-x64-setup.exe`。用户双击后按安装向导完成安装，安装后可从桌面快捷方式或开始菜单启动 `ovO`。

如果对方没有安装权限，可以发送 `release/windows/ovO-0.1.0-x64-portable.exe`。用户把文件放到任意文件夹后双击即可运行。

## 首次打开提示

因为当前安装包没有购买代码签名证书，Windows 可能会显示“Windows 已保护你的电脑”或类似安全提示。确认文件来源可信后，点击“更多信息”，再点击“仍要运行”。

## 使用步骤

1. 打开 `ovO`。
2. 在应用内点击登录，完成公司账号登录。
3. 输入或粘贴画布链接，加载画布资源。
4. 生成、预览或下载视频素材。

## 数据位置

应用会把登录状态和本地缓存保存在当前 Windows 用户目录下的应用数据文件夹中。卸载应用时默认不会删除这些数据，重新安装后通常仍可继续使用。

## 常见问题

- 打不开：确认电脑是 Windows 10/11 64 位，并重新下载安装包。
- 被安全软件拦截：确认安装包来自项目构建产物，再选择允许运行。
- 登录状态异常：在应用内退出登录后重新登录；必要时卸载后清理本地应用数据再安装。
